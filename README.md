# Momentum App

React Native 기반 모바일 생산성 앱

## 실행 방법

```bash
cd momentum-app
npm install
npm start
```

## 기술 스택

- React Native
- Expo
- NativeWind (Tailwind CSS for React Native)
- TypeScript

## 프로젝트 구조

```
momentum-app/
├── app/              # Expo Router 기반 페이지
│   ├── (tabs)/      # 탭 네비게이션 페이지
│   └── _layout.tsx  # 루트 레이아웃
├── components/       # 재사용 가능한 컴포넌트
├── assets/          # 이미지, 폰트 등
└── global.css       # Tailwind CSS 설정
```
